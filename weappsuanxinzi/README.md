# README
https://nervjs.github.io/taro/docs/GETTING-STARTED.html

## 安装
npm install -g @tarojs/cli
taro init myTaro

## 更新
taro update self
npm i -g @tarojs/cli@latest
taro update project

## 小程序开发
$ npm run dev:weapp
$ npm run build:weapp

## H5开发
$ npm run dev:h5
$ npm run build:h5

## 更新 Taro CLI 工具
| taro
$ taro update self
| npm
npm i -g @tarojs/cli@latest
| yarn
yarn global add @tarojs/cli@latest

## 更新项目中 Taro 相关的依赖
$ taro update project
